# 桌面工作流

## 首次启动

打开 OxideTerm 后，先看左侧活动栏。主要区域包括会话、连接池、连接监控、主机工具、图形/VNC、插件、云同步、文件管理器、通知和设置。

如果当前没有会话，建议先创建一个本地 shell 标签页。这可以先确认终端渲染、shell 集成、输入处理和主题设置都正常，再去添加远端主机。

## 活动栏

活动栏是进入应用各个页面的入口：

- 会话：创建、打开、分组和监控 SSH 工作。
- 文件管理器与 SFTP：浏览文件并管理传输。
- 连接池、连接监控和主机工具：查看连接运行时状态、资源快照、进程、容器、服务、tmux、日志、端口和指标。
- 图形/VNC：打开附着到已连接节点的视觉会话。
- 插件：管理已安装插件和插件设置。
- 云同步：查看同步状态并执行同步操作。
- 通知：查看最近的警告和错误。
- 设置：修改应用行为和供应商配置。

当某个流程变得不清楚时，先回到“会话”或“连接监控”。这些页面能看到主机是已保存、正在连接、已连接、已失效还是不可用。

## 终端面板

终端标签页可以用于本地 shell 和 SSH 会话。一个任务需要多个 shell 时，使用分屏面板。命令标记、shell 集成和终端历史属于具体面板；关闭面板不应该等同于断开保存的 SSH 主机。

对于长时间运行的任务，保留连接池或连接监控里的状态可见。重连行为跟连接运行时状态相关，不只取决于当前有没有可见终端标签页。

常见面板组织方式：

- 无关任务使用不同标签页。
- 需要并排比较的命令使用分屏面板。
- 一个面板持续看日志，另一个面板执行编辑或部署步骤。
- 只关闭不再需要的面板或标签页，不要误删保存连接配置。

终端右键菜单和命令栏用于明确的面板动作，例如复制、粘贴、搜索、命令选择和终端原生文件传输。背景图片和终端图像 placement 是视觉/渲染状态；如果全屏 TUI 退出后留下旧图像内容，优先清屏或重新打开面板，不要修改保存连接数据。

## 保存连接

经常复用的主机应该保存为连接。连接配置包含 host、user、port、group、color、tags、认证方式和可选的连接后命令。可以的话优先使用 SSH agent 或密钥认证。

分组用于导航和批量整理，不要把凭据或环境密码编码进分组名。

保存连接后，从“会话”视图打开它。如果连接失败，编辑原保存连接，不要创建一堆主机或标签几乎相同的重复连接。

## 连接运行时视图

连接池和连接监控显示实时运行时状态。终端像是卡住、SFTP 无法读取目录、转发无响应或需要检查重连行为时，优先看这些页面。

运行时状态和保存配置回答的是不同问题：

- 保存配置：OxideTerm 应该连接哪个主机？
- 运行时节点：这个主机当前是否连接或正在重连？
- 终端会话：哪个可见 shell 附着在运行时上？
- SFTP 会话：文件浏览是否正在使用在线传输？
- 主机工具快照：最近一次资源采样观察到了什么？
- 图形/VNC 会话：视觉查看器是否连接到节点拥有的在线会话？

主机工具适合只读式主机检查。会破坏或改变主机状态的动作应保持显式，并在执行前检查应用确认输出。

## 文件管理器与 SFTP

文件管理器用于远端浏览、上传、下载、预览和基础文件操作。远端编辑就是实际远端写入：关键文件先备份，覆盖前确认路径。

连接不稳定时，先暂停大文件传输，重连后再重试。保存连接状态和传输状态是分开的；传输失败不应该要求删除连接。

## IDE 工作区

IDE 工作区适合项目式远端编辑。先从已连接节点打开 IDE，选择远端目录，然后在 IDE 页面里的文件标签页中工作。

保存重要修改前，确认连接仍然健康。未保存编辑器缓冲区属于 IDE 工作区；关闭 IDE 标签页前，应先保存、丢弃或明确保留这些修改。

## AI 侧边栏

当当前终端、连接、文件或设置上下文很重要时，使用 AI 侧边栏。提问前保持相关标签页打开。启用工具调用后，对写入、终端输入和危险命令的审批提示要逐条检查。

执行命令时，最好让 AI 指定明确的保存连接、SSH 节点、终端会话、SFTP 会话或 IDE 工作区。不要让它从命令文本里猜主机。

## 设置

设置按功能区组织。外观、终端行为、AI 供应商、云同步、便携运行时、帮助与关于等交互式配置优先用桌面界面修改。

脚本化或可重复的修改使用 CLI，并先加 `--dry-run`。CLI 和桌面应用读取同一套配置文件。

## 命令面板与导航

日常导航使用标签页和活动栏。知道动作名称但不想离开键盘时，使用命令面板。

如果某个页面打开到了错误上下文，回到“会话”视图，选择正确连接或标签页，然后再从那里打开目标页面。

提权提示或 modem 传输这类上下文敏感辅助动作，需要先激活目标终端面板。辅助动作应作用于活跃面板/session，而不是从 prompt 文本、标签标题或保存主机标签反推目标。

## 更新

在“设置 → 帮助与关于”里查看当前版本和更新通道。稳定版、测试版和 GPUI 预览版使用独立更新通道，应该选择与当前安装包匹配的通道。
