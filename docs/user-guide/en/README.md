# OxideTerm User Guide

OxideTerm Native is a Rust/GPUI SSH workspace with terminal, SFTP, forwarding, settings, cloud sync, plugins, and a standalone CLI companion.

## Guides

- [Getting Started](./getting-started.md): install shape, configuration locations, profiles, and safe first checks.
- [Desktop Workflows](./desktop.md): terminal panes, connections, side panels, settings, and update checks.
- [CLI Companion](./cli.md): common headless workflows for diagnostics, settings, connections, backups, cloud sync, and automation.
- [Connections and Forwards](./connections-and-forwards.md): saved SSH profiles, groups, validation, and port forwarding.
- [Cloud Sync and Backups](./cloud-sync-and-backups.md): remote sync, rollback backups, restore plans, and support bundles.
- [Portable `.oxide` Bundles](./portable-oxide.md): encrypted import/export for moving profiles between machines.
- [Plugins and Secrets](./plugins-and-secrets.md): plugin state, plugin settings, and secret-safe CLI workflows.
- [Troubleshooting](./troubleshooting.md): diagnostics, strict checks, completion, and recovery patterns.
